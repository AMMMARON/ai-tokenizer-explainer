# AI Tokenizer Explainer

An interactive educational tool designed to help students understand how Large Language Models (LLMs) process text.

## Features
- **Visual Tokenizer:** Real-time visualization of how text is split into tokens.
- **Color Coding:** Each token is distinctly highlighted to show boundaries.
- **Educational Content:** Simple explanations of why tokenization matters for AI.

## How to use
Simply open `index.html` in any modern web browser. Type text into the input field to see it tokenized instantly.

## Why this matters
Tokenization is the first step in the natural language processing pipeline. By understanding tokens, students can better grasp concepts like context windows, vocabulary limits, and how models handle different languages.
